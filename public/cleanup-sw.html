🧹 Service Worker Cleanup Tool

AgentFlow Pro - Unregister all Service Workers and clear caches

Step 1: Unregister Service Workers

This will remove all registered Service Workers from localhost:3002

Step 2: Clear All Caches

This will delete all cached data

Step 3: Clear Storage

This will clear all storage including cookies, localStorage, etc.

Step 4: Restart

After cleanup, restart the application

Manual Method (Browser Console)

Alternatively, open DevTools (F12) → Console and paste:

// Unregister all Service Workers
navigator.serviceWorker.getRegistrations().then(registrations => {
  registrations.forEach(reg => reg.unregister());
  console.log('✅ Service Workers unregistered:', registrations.length);
});

// Clear all caches
caches.keys().then(names => {
  names.forEach(name => caches.delete(name));
  console.log('✅ Caches cleared:', names.length);
});

// Clear all storage
navigator.storage.estimate().then(estimate => {
  console.log('📊 Storage usage:', estimate.usage, 'bytes');
});

console.log('✅ Cleanup complete! Refresh the page.');
