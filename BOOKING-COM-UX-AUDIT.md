# AgentFlow Pro - Booking.com Style UX Audit

## 🎯 Cilj

Narediti AgentFlow Pro **preprost kot Booking.com** za navadne uporabnike.

---

## 📊 Analiza Trenutnega Stanja

### ✅ Kar Že Deluje (Kot Booking.com):

```
✓ Onboarding proces (3 koraki)
✓ Jasni gumbi za glavne akcije
✓ Progress indicatorji
✓ Quick actions na dashboardu
```

### ❌ Kar Je Preveč Tehnično:

```
✗ "API Keys" v nastavitvah
✗ "GEO/AEO" izrazi
✗ "MCP Builder"
✗ "Workflow Builder"
✗ "Agent runs"
✗ "Usage Alerts"
```

---

## 🎯 Booking.com Principi

### 1. **Preprosti Naslovi**

**Booking.com:**
- "Dodaj nastanitev"
- "Ustvari rezervacijo"
- "Poglej koledar"

**AgentFlow Pro (zdaj):**
- ❌ "Generiraj vsebino z AI"
- ✅ "Napiši opis" (better)

**Predlog:**
- "✍️ Napiši opis"
- "📧 Pošlji email"
- "🌐 Ustvari stran"

---

### 2. **Brez Tehničnega Žargona**

**Booking.com NIKOLI ne reče:**
- ❌ "API integration"
- ❌ "Database configuration"
- ❌ "SEO optimization"

**Ampak reče:**
- ✅ "Dodaj fotografije"
- ✅ "Nastavi cene"
- ✅ "Opiši nastanitev"

---

### 3. **Jasni Naslednji Koraki**

**Booking.com:**
```
[Prejšnji korak] [Naprej →]
```

**AgentFlow Pro:**
```
[← Nazaj] [Naprej →]
```

## 🎯 User Flow (Kot Booking.com)

### Nov Uporabnik:

```
1. Prijava
   ↓
2. "Dobrodošel! Kaj želiš narediti?"
   - [Dodaj nastanitev]
   - [Ustvari rezervacijo]
   - [Preskoči]
   ↓
3. Če "Dodaj nastanitev":
   - Ime in lokacija
   - Tip nastanitve
   - Število sob
   - Cene
   ↓
4. "Končano! Zdaj lahko:"
   - [Ustvari prvo rezervacijo]
   - [Dodaj fotografije]
   - [Na dashboard]
```
