# AgentFlow Pro - Chrome Extension

Generate content in context. Select text on any page and create blog posts, social copy, or emails.

## Setup

1. **Icons**: Add `icon16.png`, `icon48.png`, and `icon128.png` to this folder (or use placeholder PNGs from [placeholder.com](https://placeholder.com) – 16x16, 48x48, 128x128).

2. **Load in Chrome**:
   - Open `chrome://extensions`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select this folder

3. **Configure**:
   - Right-click the extension icon → Options
   - Set API Base URL (e.g. `https://agentflow-pro-seven.vercel.app`)
   - Set your API key (from AgentFlow Pro → Settings → Public API)

## Usage

1. Select text on any webpage
2. Click the AgentFlow Pro extension icon
3. Enter a prompt (optional) or click "Generate from selection"
4. Copy the generated content

## Manifest v3

This extension uses Manifest v3 and is ready for Chrome Web Store submission after adding icons.
