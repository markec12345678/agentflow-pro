# Beta Customer Outreach Guide

## Overview

- **Target:** 10–20 beta customers
- **Prospects to contact:** 50
- **Pricing:** 50% discount for 3 months
- **Templates:** [beta-customer-outreach-templates.md](../beta-customer-outreach-templates.md)

## Outreach Sequence

1. **Day 1–2:** Finalize list, personalize Template 1 (Initial Invitation)
2. **Day 3–4:** Send to 50 prospects
3. **Day 5:** Follow-up (Template 4) to non-responders
4. **Day 8:** Second follow-up (Template 5)
5. **Day 10:** Last chance (Template 6 if available)

## Beta Onboarding Checklist (per customer)

- [ ] Demo call scheduled and completed
- [ ] Account created (Pro plan, 50% off)
- [ ] Onboarding session (30 min)
- [ ] Tourism workflows configured
- [ ] Success metrics agreed (e.g. hours saved, content generated)
- [ ] Check-in scheduled (Day 7, Day 14)

## CRM Tracking

Track in spreadsheet or CRM:

- Company name
- Contact name, email
- Outreach date
- Response (Yes / No / Pending)
- Demo date
- Signed (Y/N)
