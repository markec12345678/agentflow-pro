## 🆘 TROUBLESHOOTING

### **Common Issues**

**Issue:** Application form won't save
```
Solution: 
- Copy all answers to a document first
- Complete in one session
- Use stable internet connection
```

**Issue:** Missing property examples
```
Solution:
- Use placeholder properties for now
- Note: "Pilot properties being onboarded"
- Can update later during review
```

**Issue:** Technical questions unclear
```
Solution:
- Use templates provided
- Be honest about experience level
- Emphasize commitment to certification
```

**Issue:** Company registration questions
```
Solution:
- Find AJPES registration at: https://www.ajpes.si/
- Use exact legal name from registration
- If no VAT yet, note: "Startup - not yet registered"
```
