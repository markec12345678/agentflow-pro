# "Ustvari Vsebino" - Kaj Narediti Ko Nimaš API Keys

## 🎯 Problem

Ko klikneš "Generiraj", vidiš samo:
- ❌ Tehnične GEO/AEO podatke
- ❌ Ni dejanske vsebine
- ❌ Ni gumba "Uporabi" ali "Shrani"

**Razlog:** Sistem rabi **OpenAI API key** za avtomatsko generiranje.

---

## ✅ Rešitev (Brez API Keys):

### Option 1: Kopiraj v ChatGPT (Najhitreje!)

```
1. Klikni "📋 Kopiraj" na dnu strani
2. Odpri ChatGPT (chat.openai.com)
3. Prilepi prompt
4. ChatGPT napiše vsebino
5. Kopiraj nazaj v AgentFlow
6. Uporabi!
```

**Čas:** 2 minuti
**Cena:** Brezplačno (ChatGPT free tier)

---

### Option 2: Dodaj OpenAI API Key

```
1. Pojdi na Settings
2. Klikni "Get key" pri OpenAI
3. Prijavi se na platform.openai.com
4. Ustvari API key (začne se s "sk-...")
5. Kopiraj key
6. Vrni se v AgentFlow → Settings
7. Prilepi key
8. Klikni "Save"
9. Vrni se na "Ustvari vsebino"
10. Klikni "🔄 Generiraj novo"
```

**Čas:** 5 minut
**Cena:** OpenAI free trial ($5 brezplačno)
