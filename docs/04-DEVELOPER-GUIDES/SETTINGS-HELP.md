# Settings - Navodila za Uporabnike

## 🎯 Pomembno za Navadne Uporabnike

### Te nastavitve NISO obvezne!

Večina uporabnikov **NE potrebuje** teh nastavitev. Sistem deluje popolnoma brez API keys.

---

## ✅ Kaj Rabiš (Minimum):

### 1. OpenAI API Key (OPCIONALNO)
```
Samo če želiš uporabljati AI funkcije:
- Avtomatsko generiranje opisov
- AI emaili za goste
- Avtomatska vsebina

Če NE želiš AI → Pusti prazno in preskoči!
```

### 2. Vse Ostalo (NE RABIŠ)
```
❌ Firecrawl - samo za web scraping (napredno)
❌ Context7 - samo za developere
❌ SerpAPI - samo za Google search (napredno)
❌ GitHub - samo za deployment (napredno)
❌ Vercel/Netlify - samo za hosting (napredno)
❌ Mailchimp - samo za email marketing (napredno)
❌ LinkedIn/Twitter - samo za social media (napredno)
❌ HubSpot/Salesforce - samo za CRM (napredno)
```

---

## 📋 Preprosta Navodila:

### Korak 1: Ignoriraj Vse
```
Sistem deluje brez API keys!
→ Dodaj nastanitev
→ Ustvari rezervacijo
→ Pošlji email ročno
```

### Korak 2: (Opcionalno) OpenAI
```
Če želiš AI funkcije:
1. Klikni "Get key" pri OpenAI
2. Prijavi se na platform.openai.com
3. Ustvari API key
4. Kopiraj (začne se s "sk-...")
5. Prilepi v polje
6. Klikni "Save"
```

### Korak 3: Končano!
```
Vrni se na dashboard in začni uporabljati.
```

## 🚀 Kaj Narediti Zdaj:

### Option A (Priporočam):
```
1. Ignoriraj to stran
2. Vrni se na dashboard
3. Klikni "Dodaj nastanitev"
4. Začni uporabljati
```

### Option B (Če želiš AI):
```
1. Nastavi samo OpenAI key
2. Vse ostalo pusti prazno
3. Klikni "Save"
4. Vrni se na dashboard
```

### Option C (Za Developere):
```
1. Nastavi vse API keys
2. Konfiguriraj integracije
3. Shrani
4. Uporabljaj napredne funkcije
```

## ❓ Pogosta Vprašanja:

### Q: Ali moram nastaviti API keys?
**A: NE!** Sistem deluje brez.

### Q: Kaj se zgodi če ne nastavim OpenAI?
**A: Nič! Vse funkcije delujejo, samo AI avtomatizacija ne.**

### Q: Kje dobim OpenAI key?
**A: platform.openai.com/api-keys (brezplačno za testiranje)**

### Q: Ali so API keys brezplačni?
**A: OpenAI ima brezplačen trial. Ostali so opcionalni.**
