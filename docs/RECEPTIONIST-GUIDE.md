# 📋 Receptionist Guide - AgentFlow Pro

## 🎯 Receptionist Role

The receptionist is the **key operator** in the AgentFlow Pro system. Responsible for **daily reservation management** with **95% automation** and only **5% manual intervention** for exceptions.

---

## 📅 Daily Tasks (5-10 minutes)

### 🏨 1. Today's Reservations Overview
**URL**: `/reservations`

**What to check**:
- ✅ **Arrivals** - Number and times of arrivals today
- ✅ **Departures** - Number and times of departures today  
- ✅ **Special Requests** - Guest special requests
- ✅ **Room Status** - Room occupancy for today

**Quick Commands**:
- **Ctrl+F** → Quick search by name or date
- **Ctrl+A** → Select all for bulk actions
- **Space** → Quick approve (if content is OK)
- **Enter** → Open details of selected reservation

---

### 🔍 2. Pending Reservations
**URL**: `/reservations/pending`

**What to check**:
- ✅ **Auto-Approval Queue** - Reservations waiting for approval
- ✅ **Exception Cases** - Manual reviews if intervention needed
- ✅ **Risk Assessment** - High-risk reservations

**Processing Priority**:
1. **🟢 Low Risk** → Auto-approve (instant)
2. **🟡 Medium Risk** → Quick review (1-2 minutes)
3. **🔴 High Risk** → Director approval (manual intervention required)

---

### 👥 3. Guest Management
**URL**: `/guests`

**What to check**:
- ✅ **Active Guests** - Currently checked-in guests
- ✅ **Special Notes** - Allergies, special requests, notes
- ✅ **Communication History** - Previous contacts with guest
- ✅ **Loyalty Status** - Repeat guests, VIP status

**Quick Actions**:
- **Click on guest** → Open profile with all data
- **Message icon** → Send message immediately
- **Phone icon** → Call directly
- **Email icon** → Send email

---

## 🔄 Reservation Processing Flow

### 📥 4. New Reservation Processing
**Standard Flow** (95% cases):
1. **Receive reservation** → System automatically checks availability
2. **Validate data** → Automatic payment card and guest ID validation
3. **Confirm in 1 click** → If everything is OK, instant confirmation
4. **Send confirmation** → Automatic email with all details

**Exception Handling** (5% cases):
1. **High Value** (>€2000) → Director approval
2. **Last Minute** (<24h) → Manual review
3. **Blacklisted Guest** → Security check
4. **Full House** (>90% occupancy) → Alternative offer
5. **Unusual Pattern** → Fraud detection

---

## 🏨 Daily Scenarios

### ☀️ Morning Routine (8:00-10:00)
1. **Check arrivals** - Guests arriving today
2. **Prepare rooms** - Cleaning status, special requests
3. **Confirm check-ins** - Automatic or manual for special cases
4. **Process special requests** - Extra bedding, children's beds

### 🌞 Afternoon Routine (14:00-16:00)
1. **Check new reservations** - Monitor new arrivals
2. **Process pending approvals** - Approve pending reservations
3. **Handle departures** - Organize guest departures
4. **Update room status** - Real-time occupancy update

### 🌙 Evening Routine (20:00-22:00)
1. **Final check-outs** - Ensure all departures are processed
2. **Prepare tomorrow** - Prepare tomorrow's arrival list
3. **Daily report** - Review today's business
4. **System backup** - Daily data backup

## 🚨 Critical Situations

### When to Escalate to Manager:
- 🔴 **System Down** → Immediately call IT support
- 🔴 **Payment Failure** → Process manually, notify guest
- 🔴 **Overbooking** → Solve immediately, offer alternative
- 🔴 **Security Issue** → Follow protocol, notify security

### Emergency Protocols:
1. **Stay Calm** → Maintain professionalism
2. **Document Everything** → Record all actions
3. **Communicate Clearly** → Inform all involved parties
4. **Follow Up** → Confirm resolution with guest
